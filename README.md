## 🚀 Deployment

This project is automatically deployed to **GitHub Pages** using GitHub Actions.

### Live Site
Your site will be available at: `https://Samsudeen22.github.io/workdone/`

### How It Works

1. **Automatic Build & Deploy**: Every time you push to the `main` branch, GitHub Actions automatically:
   - Builds your project
   - Deploys it to GitHub Pages
   - Makes it live on the internet

2. **View Deployment Status**: 
   - Go to your repository
   - Click on "Actions" tab to see build history
   - Check deployment progress in real-time

### Making Changes

Simply edit your `index.html` or `index.css` and push to `main` - your changes will be live within seconds!

```bash
git add .
git commit -m "Update: your changes"
git push origin main
```
